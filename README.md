# MSIS3363- Web Application Development

Create web pages and a multi-page application using Microsoft Visual Studio, ASP.NET, C#, HTML, HTML5, CSS3, and Microsoft SQLServer. 

Create and apply styles using, predominantly, external style sheets.

Creative Thinking; Business Knowledge and Competency;

Using the ASP.NET Web Forms Designer and tools, create visually appealing and functional web pages.

Describe and implement standard server and validation controls within the web applications.

Create a SQL Server database to support the functionality of custom web application.

Connect web pages to a SQL Server database to retrieve information into server controls and insert information into the database.

Utilize the testing and debugging tools within the ASP.NET framework.
